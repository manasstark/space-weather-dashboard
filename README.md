# Space Weather Dashboard

## Project Overview

Space Weather Dashboard is a data science and machine learning project that collects, analyzes, visualizes, and predicts space weather data using publicly available NOAA datasets.

The goal of this project is to learn:

* Python
* Data Analysis
* APIs
* Data Visualization
* Machine Learning
* Software Development Best Practices

while building a real-world portfolio project.

---

## Project Goals

* Fetch real-time space weather data from NOAA APIs
* Analyze solar flare activity
* Analyze solar wind data
* Monitor geomagnetic storm activity (KP Index)
* Create interactive visualizations
* Build a web dashboard using Streamlit
* Develop machine learning models for forecasting
